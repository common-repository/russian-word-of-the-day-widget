Russian Word of the Day Widget

== Description ==

Learn Russian words and phrases with Innovative Language's Russian Word of the Day Widget! Learn a new Russian word everyday 
with these mini-Russian lessons which include native-Russian audio, romanization, images and Russian phrases. This Word of the 
Day Widget will have you speaking Russian like a pro in no time.

== Installation ==

1. Upload `russian_wotd_widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance:Widgets page and Add the "Russian Word of the Day" Widget.

To configure:

Go to Settings -> Russian Word of the Day Widget
